# 카피 — 에러 · 빈 상태 · 시스템 메시지

> 톤: 라이더 탓으로 들리지 않게. 무엇이 일어났고, 무엇을 하면 되는지 한 줄. CTA는 행동 동사.

## 1. 로그인 / 인증 에러

### 카카오 로그인 실패
- **제목**: `로그인이 안 됐어요`
- **본문**: `카카오 로그인 중 문제가 생겼어요. 잠시 후 다시 시도해 주세요.`
- **CTA**: `다시 로그인하기`

### 카카오 로그인 취소(사용자 중단)
- **본문**: `로그인을 취소했어요. 시작하려면 다시 눌러 주세요.`
- **CTA**: `카카오로 시작하기`

### 인증번호 불일치
- **본문**: `인증번호가 맞지 않아요. 문자로 받은 6자리를 다시 확인해 주세요.`
- **CTA**: `다시 입력하기`

### 인증번호 만료
- **본문**: `인증번호가 만료됐어요. 새 번호를 받아 주세요.`
- **CTA**: `인증번호 다시 받기`

### 인증 시도 초과
- **본문**: `여러 번 틀려서 잠시 막아 뒀어요. {n}분 후에 다시 시도해 주세요.`
- **CTA**: `잠시 후 다시 시도`

## 2. 데이터 없음 (빈 상태)

### 오늘 기록 없음
- **제목**: `아직 오늘 기록이 없어요`
- **본문**: `첫 콜을 받으면 여기에 실적이 채워져요. 기록은 1분마다 자동으로 새로워져요.`
- **CTA**: `새로고침`

### 주/월 기록 없음
- **본문**: `이번 {기간}엔 아직 기록이 없어요.`

### 특정 지표만 0
- **문구**: `이 기간엔 {지표} 기록이 없어요` *(0건도 정상임을 안심시킴, 오류 아님)*

### 계정은 연결됐으나 데이터 동기화 전
- **본문**: `기록을 불러오는 중이에요. 잠깐만 기다려 주세요.`

## 3. 세션 / 연결 에러

### 세션 만료
- **제목**: `다시 로그인이 필요해요`
- **본문**: `안전을 위해 로그인이 만료됐어요. 카카오로 다시 로그인하면 이어서 볼 수 있어요.`
- **CTA**: `다시 로그인하기`

### 네트워크 오류
- **제목**: `연결이 불안정해요`
- **본문**: `인터넷 연결을 확인하고 다시 시도해 주세요.`
- **CTA**: `다시 시도`

### 데이터 불러오기 실패(서버)
- **제목**: `잠깐 문제가 생겼어요`
- **본문**: `기록을 불러오지 못했어요. 잠시 후 다시 시도해 주세요.`
- **CTA**: `다시 시도`

### 갱신 지연(스크래퍼 지연)
- **문구(배너)**: `데이터 갱신이 평소보다 늦어지고 있어요. 곧 새 기록이 들어와요.`

## 4. 권한 / 접근

### 본인 데이터 아님 / 접근 불가
- **제목**: `볼 수 없는 기록이에요`
- **본문**: `내 기록만 볼 수 있어요. 문제가 계속되면 문의해 주세요.`
- **CTA**: `내 대시보드로`

## 5. 공통 폴백 (알 수 없는 오류)
- **제목**: `예상치 못한 문제가 생겼어요`
- **본문**: `잠시 후 다시 시도해 주세요. 계속되면 문의해 주세요.`
- **CTA(주)**: `다시 시도` / **CTA(보조)**: `문의하기`
