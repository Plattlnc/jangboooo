# docs/copy — 핵심 카피 (SSOT)

> 소유: cmo. frontend는 이 문서의 한국어 문구를 **그대로** 사용한다(임의 수정 금지, 변경은 cmo 경유).
> 보이스/금지표현 근거: `docs/brand/branding-guide.md`.

## 적용 원칙
- 톤 우선순위: **신뢰 > 친근 > 동기부여**. 존댓말 기반 부드러운 톤("~해요").
- CTA는 **행동유도 동사**로 끝낸다(시작하기, 로그인, 다시 시도).
- 서비스명 자리표시자 `{서비스명}` — 네이밍 확정 시 일괄 치환.
- 전문용어 첫 등장 시 괄호 풀이. 이후 약어 허용.

## 파일
| 파일 | 범위 |

| `auth.md` | 카카오 로그인, 본인인증(휴대폰), 계정 연결 |

| `errors.md` | 로그인 실패, 데이터 없음, 세션 만료 등 에러 |
