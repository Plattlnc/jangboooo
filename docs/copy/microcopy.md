# 카피 — 마이크로카피 (버튼 · 토스트 · 공통)

> 짧을수록 좋다. 버튼은 동사로.

## 공통 버튼
| 용도 | 문구 |
|------|------|
| 시작/로그인 | `카카오로 시작하기` |
| 확인 | `확인` |
| 취소 | `취소` |
| 다시 시도 | `다시 시도` |
| 새로고침 | `새로고침` |
| 닫기 | `닫기` |
| 문의 | `문의하기` |
| 뒤로 | `뒤로` |

## 토스트 (짧은 알림)
| 상황 | 문구 |
|------|------|
| 갱신 완료 | `최신 기록으로 새로워졌어요` |
| 복사됨 | `복사했어요` |
| 로그아웃됨 | `로그아웃했어요` |
| 인증번호 전송 | `인증번호를 보냈어요` |
| 저장됨 | `저장했어요` |

## 로딩 문구
- 기본: `불러오는 중…`
- 인증: `확인하는 중…`
- 동기화: `내 기록을 가져오는 중…`

## 시간 표기
- `방금 전` / `{n}분 전` / `{n}시간 전`
- 갱신: `방금 업데이트됨`

## 숫자 단위
- 건수: `{n}건` · 퍼센트: `{n}%` · 시간대: `{시작}~{끝}시` · 점수: `{n}점`(또는 `%`, uxui 확정)
