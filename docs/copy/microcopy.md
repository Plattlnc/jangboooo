# 카피 — 마이크로카피 (버튼 · 토스트 · 공통)

> 짧을수록 좋다. 버튼은 동사로.

## 공통 버튼
| 용도 | 문구 |
|------|------|
| 시작/로그인 | `카카오로 시작하기` |
| 확인 | `확인` |
| 취소 | `취소` |
| 다시 시도 | `다시 시도` |
| 새로고침 | `새로고침` |
| 닫기 | `닫기` |
| 문의 | `문의하기` |
| 뒤로 | `뒤로` |

## 토스트 (짧은 알림)
| 상황 | 문구 |
|------|------|
| 갱신 완료 | `최신 기록으로 새로워졌어요` |
| 복사됨 | `복사했어요` |
| 로그아웃됨 | `로그아웃했어요` |
| 인증번호 전송 | `인증번호를 보냈어요` |
| 저장됨 | `저장했어요` |
| 준비 중 메뉴 탭 | `아직 준비 중이에요. 곧 열릴게요.` |

## 로딩 문구
- 기본: `불러오는 중…`
- 인증: `확인하는 중…`
- 동기화 / 최초 로딩: `내 기록을 불러오는 중…`

## 실시간 상태 (대시보드 헤더)
- 정상: `● 실시간 업데이트 중`
- 갱신 지연: `● 갱신이 조금 늦어지고 있어요`
- 연결 끊김: `● 잠시 연결이 끊겼어요`

## 배지 / 상태 라벨
- 비활성(준비 중) 메뉴: `준비 중`

## 시간 표기
- 상대시간: `방금 전` / `{n}분 전` / `{n}시간 전`
- 절대 날짜·시각(대시보드 헤더): `26년 6월 28일 (일) 오후 2:08`
  - 포맷: `YY년 M월 D일 (요일) 오전/오후 h:mm`

## 숫자 단위
- 건수: `{n}건` · 퍼센트: `{n}%` · 시간대: `{시작}~{끝}시`
- ~~점수: `{n}점`~~ — **SLA 점수 제거(2026-06-28)**. 메인 지표는 수락률 `%`.
